##  게임 소개
- **한 줄 요약:** 장애물을 피해 최대한 멀리 달리는 2D 횡스크롤 러닝 게임
- **핵심 루프:** 점프(조작) ➔ 음식 획득(회복) ➔ 점수 기록 ➔ 난이도 상승
- **핵심 재미:** 갈수록 빨라지는 속도감과 체력 3칸이 주는 아슬아슬함

##  주요 기능
- **다이내믹 액션:** 2단 점프 및 무적 아이템(황금 츄르)
- **인터랙티브 UI:** 체력에 따라 UI 고양이 이미지가 실시간으로 변화
- **도전 욕구 자극:** 최고 점수 저장 및 게임 속도 가속 시스템
